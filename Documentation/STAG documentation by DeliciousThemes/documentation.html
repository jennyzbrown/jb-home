Intro

This is not your ordinary template. Stag is about the extraordinary. Both design and simplicity astound in this minimalist-meets-versatile HTML template. Stag is packed with the features you`ll ever need to build a professional, yet modern-looking website for your business. Stag’s user experience is seamless, and provides a breathtaking foundation for freelancers, agencies, photographers, bloggers, and anyone in search of a brilliant web presence. Its element options are vast, and were carefully designed, crafted, and positioned with the utmost attention to detail throughout.

Files and Folders

 

All html pages are sugestively named, so you know what page is for.

  1. assets/css/color-schemes - styles for each color scheme
  2. assets/fonts - the icon fonts Fontawesome and Et Line fonts
  3. assets/images - images used in the template. The Folder includes main images used in html layout, but also some subfolders:
    • portfolio - keeps thumbnails used for portfolio items
    • blog - keeps thumbnails used for blog items
    • clients - keeps images used for the clients section
    • team - keeps images used for the team section
    • bg - keeps images used for the parallax backgrounds
  4. assets/js - javascript files folder
  5. php - php files folder
  6. revslider - this folder holds the jQuery Revolution plugin files and assets
  7. style.css - holds the styling code of the template

HTML

The template is based on a fluid, responsive layout and built using HTML5, CSS3 and JavaScript.

The files which contains all the HTML structure are the ones with the .html suffix.

You can change the text from the template by navigating through each file and locate each line of text you want to change.

To change the logo, for example, open each html page, locate the line which contains the address to the current logo(assets/images/logo.svg) and change it with the address of your own logo.

He​ader

Stag comes with multiple header styles for both initial and on scroll states. Most of the header setups are made in assets/js/custom-navscroll.js file.

 

Header settings

Open assets/js/custom-navscroll.js and adjust the variables for your needs(ex: adjust padding values): 
var initial_header_color = '#fff',
	fallback_header_color = '#fff',
	onscroll_header_color = 'rgba(255, 255, 255, 0.8)',
	initial_padding_top = '48px',
	initial_padding_bottom = '48px',
	onscroll_padding_top = '16px',
	onscroll_padding_bottom = '16px',
	logo_width = '85px',
	logo_height = '25px',
	logo_onscroll_height = '25px';

There are a couple of pages with custom header settings. Ex: homepage-fullscreen-intro.html

At the end of the style.css file you`ll find a section called "header custom styles for different pages". This is for changing the initial and onscroll header background colors and also the header background color for when the menu is triggered on mobile:

#header.initial-state {
    background: rgba(255, 255, 255, 0) !important;
}
#header.scrolled-header {
    background: rgba(255, 255, 255, 0.9) !important;
}
@media only screen and (max-width: 1023px) {
    #header.initial-state.is-triggered {
        background: #323232 !important;
    }            
}

At the end of some files(ex: homepage-fullscreen-intro.html), you`ll find a call to a file located at "assets/js/custom-header.js" for changing the initial/onscroll logo and also the initial/onscroll color scheme:

var stag_header_logo_about_me = jQuery(".about-me #header .logo img");
var stag_schemes_about_me = jQuery(".about-me .main-navigation, #headersocial, .bm");
jQuery(window).scroll( function() {
    var stag_val = jQuery(this).scrollTop();
    if ( stag_val > 150 )  {
        stag_header_logo_about_me.attr("src","assets/images/logo.svg");
        stag_schemes_about_me.removeClass('dark-header').addClass('light-header');
    }
    else {
        stag_header_logo_about_me.attr("src","assets/images/logo-white.svg");
        stag_schemes_about_me.removeClass('light-header').addClass('dark-header');
    }
});

To change the initial color scheme of the header, set the dark-header or light-header class to the elements of the header: main-navigation and bm(burger menu icon): 

<nav id="site-navigation" class="main-navigation minimal-menu dark-header" role="navigation">

...
<div class="bm dark-header minimal-menu">

How to Set Up the One Page menu

The one page menu is located inside the homepage-parallax-one-page.html file and it looks like this:

<nav id="site-navigation" class="main-navigation classic-menu light-header" role="navigation">
    <div class="dt-onepage-menu-container">
        <ul id="primary-menu" class="menu">
            <li class="initial menu-item"><a href="#hello">HELLO</a>
            </li>
            <li class="menu-item"><a href="#work">WORK</a>
            </li>
            <li class="menu-item"><a href="#process">PROCESS</a>
            </li>
            <li class="menu-item"><a href="#journal">JOURNAL</a>
            </li>
            <li class="menu-item"><a href="#contact">CONTACT</a>
            </li>
            <li class="external menu-item"><a href="homepage-revolution-slider.html"><i class="fa fa-external-link"></i> DEMO</a>
            </li>
        </ul>
    </div>
</nav>
As you can notice, there are some special classes used inside the menu:
 
- initial - this class is used for the li element and makes the menu item highlighted. 
- external - this class is used to indicate an external link, which doesn`t belong to the click-and-scroll menu functionality. Use it everytime you have an outside page. 
 
To create a scrolling menu element, you need to use an URL which contains a link to a section from the page, identified by an ID. For example: #about will scroll down to "About" section. Make sure to set the id to the About section, like this: <section id="about">

How to Set Up the Homepage Portfolio

The homepage portfolio can use a regular, mosaic, parallax or masonry layout and each portfolio thumbnail can have 2 sizes: small thumbnail(identified by "item-small" class), wide thumbnail(identified by "item-wide" class).

For best results, use images bigger than or at least 640x480px. 

The thumbnail images are located inside the "portfolio" folder from "assets/images".

The portfolio can be filterable or not. To make it not filterable, remove the list with the id "filters" from the page. You can assign a category to an item element, so it can be sorted. Just follow the pattern used inside the portfolio section.

Parallax Backgrounds

You can set parallax backgrounds for sections. The changes you need to make in order to obtain a parallax background are located at the HTML and jQuery levels. 

1. HTML

You need to give to your section the "parallax-section" class and another class or ID for individualizing it. You can also set a transparent cover for it(using a div with classes "darker-overlay" or "lighter-overlay"). For example, the HTML section for the testimonials looks like this:

<section id="testimonials" class="delicious-section parallax-section lazy"  data-original="assets/images/bg/testimonials-bg.jpg">
    <div class="darker-overlay">
        <div class="container">
            ...
        </div>
        <!-- end container -->
    </div><!--end darker-overlay-->
</section>

 

You`ll find at the end of the style.css file a section called "custom styles for sections" which you could use to further customize these sections.

2. jQuery

Last step is to assign the ID or class to the Parallax script mechanism using jQuery. You can add it inside "assets/js/custom-parallax.js" at "Parallax Sections". The code for the quote section looks like this:

$('#testimonials').parallax("50%", 0.4);
The options are:
Position - Horizontal position of the element(50%).
Inertia - speed to move relative to vertical scroll. Example: 0.1 is one tenth the speed of scrolling, 2 is twice the speed of scrolling.
 

The backgrounds are lazy loaded, this being a performance feature of the template.

How to Change the Color Scheme of the Template

The template comes in 6 color schemes: yellow, orange, blue, green, red, purple.

The current theme is set to red. The code for it is located in every html page, inside the <head> </head> section:

<link rel="stylesheet" type="text/css" href="assets/css/color-schemes/green/green.css" media="screen" />

Change red from the above line with other color from the list and the template will import the styles for the respective color. For example:

<link rel="stylesheet" type="text/css" href="assets/css/color-schemes/red/red.css" media="screen" />

 

How to use Animations

You can add animations to any HTML block. As you already noticed in the live demo, there are some animations which reveal each element while you scroll down the page. 

For example, on the services.html page you`ll notice in the service div tow classes: wow and fadeInUp.:

<div class="four columns">
    <div class="dt-service-elem content-left square no-fill wow fadeInUp" style="visibility: visible; animation-name: fadeInUp;">
        <div class="dt-service-icon"><span class="icon-beaker"></span>
        </div>
        <div class="dt-service-content">
            <h4 class="dt-service-title ">WEB DEVELOPMENT</h4>
            <p>A small river named Duden flows by their place and supplies it with the necessary regelialia.</p>
        </div>
    </div>                                       
</div>

The things that make the element being animated are the classes wow and fadeInDown.

So, everytime you want to animate an element, you have to add to it the class wow and the class for the effect. The entire list of effects can be found at http://daneden.github.io/animate.css/

How to Set Up the Google Maps Widget

Stag template has integrated a Google Map inside multiple spots on the template, like the contact.html pages. This map is generated by the next HTML code: 

<div class="map-wrapper" id="delicious_map">
    <div id="google_map"></div>
</div>
 
In order to edit the map and customize it with your own location, open "assets/js/custom-map.js".
 
You need to use a set of coordinates. I used this set: 37.422117,-122.084053 . The coordinates appear for 3 times inside the file, so make sure to replace all 3 instances with your own coordinates. 
 
After that, you can customize the tooltip with your own text. Locate this code:
 
var contentString = '<div class="map-tooltip">'+
'<h6>Company Headquarters</h6>'+
'<p>Now that you visited our website, how about <br/> checking out our office too?</p>'+
'</div>';
...and replace the content with your own. 
 
That`s it! The map should use your details from now.
 

Google Maps API Key

Google recently changed their terms of service for the maps and they won`t be working on websites anymore without an API key. For this reason, the call to the Google Maps script located on the .html page at the bottom requires a key:

<script type="text/javascript" 
src="https://maps.googleapis.com/maps/api/js?key=AIzaSyAVkNtnbmfWK1GiAgmSe8q8QAGRwDbwtwo&callback=initmymap">
</script>

This is a demo key which works fine for now but in the future, I`m sure it will pass the 25000 daily usage so it`s best to create one for yourself at https://console.developers.google.com/apis/api/maps_backend/overview

Very Important:

If you get rid of the <div id="google_map"></div> code from your HTML page, make sure to delete the scripts related to google map from the bottom of the HTML page:

 
<!-- GOOGLE MAPS - remove if you don`t want to use the maps on a page -->
<script type="text/javascript" src="assets/js/custom-map.js"></script>
<script type="text/javascript" 
src="https://maps.googleapis.com/maps/api/js?key=AIzaSyAVkNtnbmfWK1GiAgmSe8q8QAGRwDbwtwo&callback=initmymap">
</script>
Otherwise, the page will get broken. Same thing for the isotope scripts/portfolio grid.
